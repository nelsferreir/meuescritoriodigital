# Meu Escritório Digital

Um sistema de gestão simplificado para advogados, permitindo o cadastro e fácil acesso a informações de clientes, casos e documentos.

## ✨ Principais Tecnologias

- **Framework**: [Next.js](https://nextjs.org/) (com App Router)
- **Backend e Banco de Dados**: [Supabase](https://supabase.io/)
- **Estilização**: [Tailwind CSS](https://tailwindcss.com/)
- **Componentes UI**: [shadcn/ui](https://ui.shadcn.com/)

## 🚀 Rodando Localmente

1.  **Instale as dependências:**
    ```bash
    npm install
    ```

2.  **Configure as Variáveis de Ambiente:**
    - Renomeie o arquivo `.env.example` para `.env.local`.
    - Preencha as chaves da API do seu projeto Supabase.

3.  **Execute o servidor de desenvolvimento:**
    ```bash
    npm run dev
    ```

Abra [http://localhost:3000](http://localhost:3000) no seu navegador para ver o resultado.
